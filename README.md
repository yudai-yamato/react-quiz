# React Quiz App

## 概要
4択クイズを解いて、最後に正解数を表示するアプリ。

## ページ一覧
1. TOPページ
2. クイズページ
3. 結果ページ

## 機能一覧
### 1. TOPページ
- スタートボタン：押したらクイズページへ遷移

### 2. クイズページ
- 問題文を表示
- 選択肢を表示
- 選択肢を押したら…
  1. 正誤判定を行い、記録する
  2. 問題が残っていれば次の問題を表示
  3. 残っていなければ結果ページへ遷移

### 3. 結果ページ
- 演出：カーテンコールを1〜2秒表示→非表示
- クラッカー演出
- 問題総数と正解数を表示

## セットアップ
```bash
npm install
npm run dev
